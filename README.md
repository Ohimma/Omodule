## 二：项目说明
```
vue3 + element-plus 

git clone https://github.com/Ohimma/Omodule.git
git remote add github git@github.com:Ohimma/Omodule.git
git remote add gitee  git@gitee.com:ohimma/omodule.git
cd Omodule
npm install
npm run serve
```
